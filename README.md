# Interpretable AI

## Tables
1. Поиск причинно-следственных связей: кто-то делает единый код для refutation. Потом каждый строит свой граф (возможно, отобрав перед этим фичи) и прогоняет модель. Затем сравниваем наши модели с помощью refutation.
2. Обучение моделей черного ящика: Выбираем алгоритмы, каждый пытается сделать на своей максимальное качество. Затем аггрегируем и сравниваем. Предлагаю модели: ElasticNet, LightGBM, RandomForest, NeuralNet, KNeighbors, LightAutoML, DecisionTree.
3. Применение SHAP: пусть каждый на своей модели запускает. А в презе просто покажем лучший метод.
4. Применение LIME: аналогично п.3.
